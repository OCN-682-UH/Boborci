This plot is considered a good plot because it follows best data practices as defined in The American Statistician-Data Organization in Spreadsheets by Broman and Woo. These practices include Good Naming Conventions, No Empty Cells, and One Item per Cell and Save Data as Plain Text Files. I also created a Data Dictionary, to ensure people can understand what exactly the data are referring to. As a visualization, this plot follows the best practices we learned in class. Some of these aspects are

  • Clean and visible plot and axis titles
  • No unnecessary colors
  • Clear legend if necessary
  • Decluttered
